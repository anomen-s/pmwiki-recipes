JavaAesCryptTest
================

This java application can test encryption rutine used in AesCrypt PmWiki recipe
( http://www.pmwiki.org/wiki/Cookbook/AesCrypt ).

This is sample of PmWiki page content:

(:aes C1bwAKkCQm2N7Yuk :)

(:aes yxR6Tg8PDw8WfHwFPR1xrg== :)

(:aes vrJ5TpeXl5eYW4yixn0= :)

(:aes 4bJ5TmRkZGRqAw== :)

(:aes /bJ5Tl9fX18Knw== :)

(:aes HLN5TqSkpKTXq3iJp/XqnA== :)


These lines are encrypted using defult password "TopSecret".

Execution:
----------
Use following command to start command-line interface:

  java -jar JavaAesCryptTest-jar-with-dependencies.jar


Note: GUI interface is not yet implemented.
